 Project Purpose:
The purpose of this repository is to complete the OpenSSF Scorecard lab assignment. This project shows course completion, repository badge integration, and security improvements based on Scorecard recommendations.

 Security Improvements
The following improvements were made to strengthen the repository:
- Enabled branch protection rules
- Added a SECURITY.md file for vulnerability reporting
- Improved repository documentation
- Applied secure development practices based on Scorecard feedback

 Task Summary
This repository supports the following lab tasks:
- **Task 1:** OpenSSF Scorecard course completion
- **Task 2:** Integration of OpenSSF Scorecard and Best Practices badges
- **Task 3:** Security fixes and Scorecard improvement efforts
